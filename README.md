# File Organizer Automation Tool

## Description

This Python automation tool organizes files automatically based on their file extensions.
It reduces manual effort and improves system-level automation.

## Features

- Automatic folder creation
- File movement based on extension
- Handles unknown file types
- Simple and scalable automation script

## Technologies Used

- Python
- OS Module
- File Handling
- Automation Scripting

## How to Run

1. Place files inside `test_files` folder
2. Run:
   python file_organizer.py
3. Files will be organized automatically
